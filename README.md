# Java Calculator

A simple calculator application built using **Java** to practice core programming
concepts such as control flow, methods, input handling, and basic arithmetic operations.

This project is intended for learning purposes and to demonstrate clean, structured
Java code.

---

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Input validation
- Handles divide-by-zero cases

---

## Tech Stack

- Language: **Java**
- JDK: Java 8 or above
- IDE: Any (Eclipse)
